# 🥗 Aplicativo de Nutrição

## 📌 Sobre o Projeto
Este projeto consiste no desenvolvimento de um sistema de gerenciamento nutricional, permitindo o cadastro e acompanhamento de dados alimentares.

---

## 🎯 Objetivo

- Gerenciar informações nutricionais
- Controlar consumo alimentar
- Auxiliar no acompanhamento de dieta
- Organizar dados de usuários e alimentos

---

## 🧱 Funcionalidades

- Cadastro de usuários
- Cadastro de alimentos
- Registro de refeições
- Consulta de dados nutricionais
- Atualização e exclusão de registros

---

## ⚙️ Tecnologias Utilizadas

- Java
- Android Studio
- Banco de Dados
- Lógica de programação

---

## 📊 Estrutura do Sistema

O sistema segue o padrão:

Interface → Processamento → Banco de Dados
